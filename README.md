## URL Parser
一个可以直接查看并编辑页面参数的Chrome插件，并可以直接进行编辑
![](https://github.com/SinanJS/url-parse/blob/master/public/images/test.png?raw=true)

npm 安装依赖
```shell
# npm
npm install
# yarn
yarn intall
```

打包编译
```shell
# npm
npm run build
# yarn
yarn build
```
**安装步骤**
1. 打开Chrome浏览器的“管理扩展程序”
2. 开启右上角的“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择生成的 build 目录文件，即可安装成功
